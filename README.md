# Eddy Charger Branded Website - Agents

This branded websites is built by [`Extension Team`](https://extension.team/) and powered by [`Svelte`](https://kit.svelte.dev/). This specific instance is designed for Teddy Benz and meant to target others interested in real estate investing, business buying, and capital raising and lending for both real estate and business acquisitions.

## Developing

Once you've created a project and installed dependencies with `npm install` (or `pnpm install` or `yarn`), build your package and start a development server:

```bash
npm run build

npm run dev

# or start the server and open the app in a new browser tab
npm run dev -- --open
```
